 Autocomplete and Autocorrect Data Analytics

Implemented autocomplete and autocorrect algorithms using 
English word frequency data.

# Dataset
- unigram_freq.csv: English words with frequency counts

# What I did
- Loaded and explored word frequency dataset
- Built autocomplete function using prefix matching
- Built autocorrect function using difflib similarity
- Visualized top 20 frequent words
- Analyzed word length distribution

# Results
- Autocomplete suggests top words for any given prefix
- Autocorrect fixes common spelling mistakes
- Most English words are 5-8 characters long

# Tools Used
Python, Pandas, Difflib, Matplotlib, Seaborn
